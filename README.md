💳 Credit Scoring Model

📌 Project Overview

This project aims to predict whether a person is creditworthy (Good Credit Risk) or not creditworthy (Bad Credit Risk) using Machine Learning techniques.
It helps financial institutions make better decisions while approving loans.


🎯 Objective

- Predict credit risk based on customer data
- Improve decision-making for loan approval
- Compare performance of different ML models


📊 Dataset

- Source: Kaggle Credit Card / Loan Prediction Dataset
- Features include:
  - Age
  - Income
  - Loan Amount
  - Payment History
  - Credit Score


🛠️ Technologies Used

- Python
- Pandas
- Scikit-learn
- NumPy


⚙️ Workflow

1. Data Collection
2. Data Cleaning (handling missing values)
3. Feature Selection
4. Train-Test Split
5. Model Training
6. Model Evaluation


🤖 Machine Learning Models Used

- Logistic Regression
- Random Forest Classifier


📈 Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC


📊 Output

- Predicts whether a person has:
  
  - ✅ Good Credit Risk
  - ❌ Bad Credit Risk

- Achieved Accuracy: ~85–90%
